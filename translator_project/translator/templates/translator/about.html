About Voice Translator

Welcome to Voice Translator, your go-to tool for seamless translation between multiple languages.

  • Translate text between multiple languages including English, Spanish, Hindi, French, German, Chinese, Japanese, etc.
  • Supports voice input for translating spoken words.
  • Responsive design using Bootstrap for optimal viewing on various devices.
  • Easy-to-use interface with intuitive controls.

The Voice Translator app utilizes a combination of Django for backend processing and Bootstrap for frontend design. Users can select their preferred languages for translation, input text or speak directly into the microphone for translation, and receive instant results.

For voice translation, the app uses the ResponsiveVoice.js library to convert text into speech in the selected target language.
